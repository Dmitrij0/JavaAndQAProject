Описание:
Данная программа является калькулятором больших целых чисел с неограниченной длиной. 
Ограничением являются настройки JVM, операционной системы и возможности hardware

Возможности:
1. Калькулятор производит вычисление сложных арифметических выражений с применением скобок любого уровня вложенности.
2. Поддерживаемые операции '+', '-', '*', '/'

Особенности:
Калькулятор реализован для собственного класса больших целых чисел
Калькулятор не поддерживает числа с плавающей точкой, поэтому опреация деления вычисляется приблизительно.

Инсталяция:
1. На машине должна быть установлена jvm-1.8.x 
2. Разархивировать каталог .\bin
3. Запустить приложение при помощи *.bat
   Можно запустить приложение прямо из командной строки (версии файла могут меняться):
   java -jar calculator-1.1.jar
